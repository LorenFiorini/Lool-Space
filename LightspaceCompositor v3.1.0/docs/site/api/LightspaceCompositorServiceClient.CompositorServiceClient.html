    Class CompositorServiceClient

    An RPC client wrapper for an ICompositor implementation. Used to control the real Compositor in a separate process.

    Parameters

    Gets the compositor parameters. This is proxied through the separate Parameters client, since the Parameters isn't a basic struct or a POCO, and ServiceWire does not do recursive proxying of objects. That is, if we want to call methods (gets and sets) on Parameters, we need to have a separate proxy set up for that.
